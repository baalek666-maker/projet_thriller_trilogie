# Projet Trilogie Thriller Psychologique

Trilogie de thrillers psychologiques au rythme addictif, dans la veine de Freida McFadden, conçue pour le marché Amazon.fr (KDP ebook + poche).

## Pitch global

Une héroïne ordinaire bascule dans un huis clos domestique où les apparences mentent, l'ennemi dort à côté d'elle, et chaque révélation la rapproche d'une vérité qu'elle aurait dû fuir.

## Méthodologie de travail

Le projet avance par phases validées, avec un commit Git par phase et push sur GitHub.

| Phase | Livrable | Dossier |
|-------|----------|---------|
| 0 | Initialisation du dépôt et du README | racine |
| 1 | Recherche marché : 3 concepts avec hook, public cible et viralité | `01_recherche/` |
| 2 | Architecture narrative globale sur 3 tomes + étapes Vogler détaillées | `02_structure/` |
| 3 | Stratégie marketing (titre, accroches, lead magnet, couverture, blurb) | `03_marketing/` |
| 4 | Écriture du Tome 1 en blocs de 10-15 chapitres | `04_manuscrits/` |

Règles internes :
- Une phase validée par l'utilisateur avant de passer à la suivante.
- Pas de manuscrit avant la stratégie marketing.
- Style : chapitres courts (3-8 pages), suspense constant, narrateur peu fiable, ambiances oppressantes.
- Objectif : 250-350 pages au format poche, fin choc du Tome 1, cliffhanger global pour la trilogie.
